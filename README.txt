										Peak Fitness E-Commerce Website




Peak Fitness is a simple e-commerce website designed with HTML, CSS, and JavaScript. It simulates an online store that promotes accessible & high-quality fitness equipment to encourage persons to be healthier and more active. It is fully functional and includes basic functions such as registration and validated login, invoice generation, and a shopping cart.

------------------




***Running the Website***

1. Ensure all the files are within the same folder.
2. Open Peak.html in any browser.
3. You may sign in using default credentials:

Username: admin
Password: admin123

4. Upon a successful login, you will be redirected to the products page where you may add or remove items from your cart as you wish.
5. To checkout, head to your cart and click Check Out to fill in payment info and billing and shipping information.
6. Finally, you can pay and generate your invoice.




**Key Functionality**

- Data Validation: Validation can be tested in the login forms and in checkout.
- Checkout Functionality: Users are able to enter their billing and payment information to confirm their order to receive an invoice.
- Invoice Generation: Upon checkout, an invoice can be printed with the details of your order.

------------------




**Tools Used**

Various tools were used to envision and build the website:


Main Framework

- HTML5 – Main markup language used to structure the content of the web pages.
- CSS3 – Styling language used to design the pages, making them more visually appealing.
- JavaScript – Scripting language used for logic and interactive functionality (login, registration, cart management).



Other Tools

- Canva – Online graphic designing tool used to create a schematic, as well as a mockup of the website.
- Boxicons – Open source library for clean, simple vector icons that enhance the UI (user, cart icons, etc.).
- Draw.io – Online diagram creating tool used to create a sitemap of the project.
